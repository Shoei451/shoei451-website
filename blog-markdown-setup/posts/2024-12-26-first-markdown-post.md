# はじめに

これは markdown で書いた最初のブログ記事です。このシステムでは、**markdown** で簡単に記事を書くことができます。

## Markdown の利点

Markdown を使うことで:

- シンプルな記法でリッチなコンテンツを作成できる
- バージョン管理がしやすい
- プレーンテキストなので軽量
- 多くのエディタでサポートされている

## コード例

JavaScript のコード例:

```javascript
function greet(name) {
    return `Hello, ${name}!`;
}

console.log(greet("World"));
```

Python のコード例:

```python
def calculate_area(radius):
    return 3.14159 * radius ** 2

print(f"Area: {calculate_area(5)}")
```

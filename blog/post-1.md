# 微分積分の基礎

微分と積分は数学の中でも特に重要な概念です。今回は基礎的な部分を整理してみます。

## 微分とは何か

微分は「瞬間的な変化率」を求める操作です。例えば、車の速度を考えてみましょう。

- 平均速度：ある区間での移動距離 ÷ 時間
- 瞬間速度：ある一点での変化率（微分）

### 導関数の定義

関数 $f(x)$ の導関数 $f'(x)$ は以下のように定義されます：

```
f'(x) = lim[h→0] (f(x+h) - f(x)) / h
```

## 基本的な微分公式

よく使う微分公式をまとめます：

| 関数 | 導関数 |
|------|--------|
| $x^n$ | $nx^{n-1}$ |
| $\sin x$ | $\cos x$ |
| $\cos x$ | $-\sin x$ |
| $e^x$ | $e^x$ |
| $\ln x$ | $1/x$ |

## 積分とは

積分は微分の逆操作です。面積を求めるときに使います。

### 不定積分

原始関数を求める操作：

```
∫ f(x) dx = F(x) + C
```

ここで、$F'(x) = f(x)$ であり、$C$ は積分定数です。

### 定積分

区間 $[a, b]$ での面積を求める：

```
∫[a→b] f(x) dx = F(b) - F(a)
```

## 実践問題

**問題**: $f(x) = x^2 + 3x + 2$ を微分せよ。

**解答**:
```
f'(x) = 2x + 3
```

各項を個別に微分して足し合わせます。

## まとめ

微分積分の基本的な考え方：

- **微分**: 瞬間的な変化率を求める
- **積分**: 累積的な量（面積など）を求める
- この2つは互いに逆の操作（微積分の基本定理）

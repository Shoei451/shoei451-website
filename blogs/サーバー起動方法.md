# 🚀 ローカルサーバーの起動方法

## 問題
`Failed to fetch` エラーは、ブラウザが `file://` プロトコルでMarkdownファイルを読み込めないために発生します。

## 解決方法

### ✅ 方法1: Python サーバー（推奨）

1. **`start-server.bat` をダブルクリック**
   - 場所: `C:\Users\user\Documents\shoei451\shoei451-website\start-server.bat`

2. **ブラウザで開く**
   ```
   http://localhost:8000/blogs/
   ```

3. **停止する時**
   - コマンドプロンプトで `Ctrl + C`

---

### ✅ 方法2: VS Code の Live Server

1. **VS Code をインストール済みなら:**
   - VS Code でプロジェクトフォルダを開く
   - 拡張機能 "Live Server" をインストール（まだなら）
   - `index.html` を右クリック → "Open with Live Server"

---

### ✅ 方法3: Node.js の http-server

Pythonがない場合:

1. **Node.js がインストール済みなら:**
   ```bash
   npm install -g http-server
   cd C:\Users\user\Documents\shoei451\shoei451-website
   http-server -p 8000
   ```

2. **ブラウザで開く**
   ```
   http://localhost:8000/blogs/
   ```

---

## 📝 使い方

### サーバー起動後

1. **ブログ一覧を見る:**
   ```
   http://localhost:8000/blogs/index.html
   ```

2. **Markdown記事を見る:**
   ```
   http://localhost:8000/blogs/markdown-blog.html?post=2024-12-26-markdown-sample
   ```

3. **テストページ:**
   ```
   http://localhost:8000/blogs/test.html
   ```

---

## ⚠️ 重要

- **必ずサーバーを起動してから** ブラウザでアクセスする
- `file://` で開くと `Failed to fetch` エラーが出る
- `http://localhost:8000/` で開けばエラーが出ない

# 📚 統合版ブログシステム - 使い方ガイド

## ✨ 統合完了！

既存のblogシステムに**Markdownサポート**を追加しました。

## 📁 ファイル構成

```
blogs/
├── index.html              # メインのブログ一覧ページ（既存のまま）
├── markdown-blog.html      # 新規：Markdown記事を表示するページ
├── posts-list.js          # 更新：HTML/Markdown両方の記事リスト
└── posts/
    ├── 2024-12-26-markdown-sample.md  # Markdown記事の例
    ├── _template.md                   # 新規記事用テンプレート
    ├── 2024-12-01-first-post.html    # 既存のHTML記事
    └── 2024-12-15-learning-biology.html
```

## 🚀 使い方

### ✅ Option 1: Markdownで書く（推奨）

**手順:**

1. **新しいファイルを作成**
   ```
   blogs/posts/2024-12-26-my-post.md
   ```

2. **frontmatter（メタデータ）を追加**
   ```markdown
   ---
   title: 記事のタイトル
   titleEN: English Title
   date: 2024-12-26
   category: Personal
   categoryJP: 日記
   ---

   # ここから本文

   Markdownで自由に書けます...
   ```

3. **posts-list.js に追加**
   ```javascript
   {
       title: "記事のタイトル",
       titleEN: "English Title",
       date: "2024-12-26",
       category: "Personal",
       categoryJP: "日記",
       excerpt: "記事の簡単な説明...",
       link: "markdown-blog.html?post=2024-12-26-my-post",  // ⚠️ .mdは付けない！
       image: "../images/favicon.png"
   }
   ```

**メリット:**
- HTMLを書かなくていい ✅
- シンプルな記法 ✅
- 書くことに集中できる ✅
- バージョン管理しやすい ✅

## 📝 Markdown記法クイックリファレンス

### 見出し
```markdown
# H1
## H2
### H3
```

### 強調
```markdown
**太字**
*斜体*
~~取り消し線~~
```

### リスト
```markdown
- 項目1
- 項目2

1. 順序1
2. 順序2
```

### リンク・画像
```markdown
[リンク](https://example.com)
![画像](../images/image.png)
```

### コード
インライン: `code here`

ブロック:
````markdown
```python
def hello():
    print("Hello!")
```
````

### 引用
```markdown
> 引用文
> - 作者
```

### 表
```markdown
| 列1 | 列2 |
|-----|-----|
| A   | B   |
```

## 🎯 ファイル命名規則

**Markdown記事:**
```
YYYY-MM-DD-title.md
```
例: `2024-12-26-my-first-post.md`

**HTML記事:**
```
YYYY-MM-DD-title.html
```
例: `2024-12-26-my-first-post.html`

## 📋 ワークフロー例

### Markdownで新記事を書く

1. `posts/_template.md` をコピー
   ```bash
   cp posts/_template.md posts/2024-12-26-new-post.md
   ```

2. frontmatterを編集
   ```markdown
   ---
   title: 新しい記事
   titleEN: New Post
   date: 2024-12-26
   category: Personal
   categoryJP: 日記
   ---
   ```

3. 本文を書く（Markdownで）

4. `posts-list.js` の**先頭**に追加（新しい記事ほど上）
   ```javascript
   const blogPosts = [
       {  // ← ここに追加
           title: "新しい記事",
           ...
           link: "markdown-blog.html?post=2024-12-26-new-post"
       },
       // 以前の記事...
   ];
   ```

5. ブラウザで確認
   - 一覧: `blogs/index.html`
   - 記事: `blogs/markdown-blog.html?post=2024-12-26-new-post`

## 🔧 トラブルシューティング

**記事が表示されない？**
- ファイル名が正しいか確認
- posts-list.js のリンクに `.md` が付いていないか確認
- frontmatterの形式が正しいか確認（`---` で囲む）

**スタイルが崩れる？**
- `markdown-blog.html` が `../css/common.css` を読めているか確認
- ブラウザのデベロッパーツールでエラーを確認

**画像が表示されない？**
- 画像パスが正しいか確認（Markdown記事から見た相対パス）
- 例: `![画像](../images/favicon.png)`

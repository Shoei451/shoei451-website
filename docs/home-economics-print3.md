# **「食事と栄養・食品衛生」テスト対策プリント**

## **1\. 食事と栄養素の基本**

このセクションでは、健康な食生活を送るための土台となる栄養学の基礎を学びます。私たちが毎日口にする食品には、生命を維持し、体を成長させるために不可欠な「栄養素」が含まれています。特に重要な5つの栄養素、すなわち「5大栄養素」のそれぞれの役割を正確に理解することが、バランスの取れた食事を考える第一歩となります。

1. 食品に含まれる成分を摂取することで生命を維持することを`（　栄養　）`といい、摂取される成分そのものを`（　栄養素　）`とよびます。  
2. 栄養素は、体内で働く役割によって、`（　炭水化物　）`、`（　脂質　）`、`（　たんぱく質　）`、 `（　無機質　）`、`（　ビタミン　）`の5つに分類され、これらをまとめて**5大栄養素**といいます。  
3. 5大栄養素のうち、特に体内でエネルギー源となる`（　炭水化物　）`、`（　脂質　）`、`（　たんぱく質　）`の3つを**3大栄養素**（エネルギー産生栄養素）といいます。  
4. 栄養素には分類されませんが、体温調節や物質の運搬など、生命維持に不可欠で、体の半分以上を構成する成分は`（　水　）`です。  
5. ダイエットなどで食物繊維を多く含む食品を過剰に摂取すると、`（　カルシウム　）`や`（　鉄　）`といった重要な無機質の吸収を阻害し、貧血などの原因となることがあります。

## **2\. 炭水化物**

このセクションでは、私たちの主要なエネルギー源である炭水化物について学びます。炭水化物にはエネルギーになる「糖質」と、便秘予防やコレステロール値の低下などに役立つ「食物繊維」の2種類があります。それぞれの働きや、米や小麦、いも類といった炭水化物を多く含む食品の特性を理解することは、日々のエネルギー管理において非常に重要です。

#### **炭水化物の種類と働き**

1. 炭水化物は、体内で消化・吸収されてエネルギーになる`（　糖質　）`と、人の消化酵素ではほとんど消化・吸収されない`（　食物繊維　）`に分けられます。  
2. 食物繊維は、小腸の`（　蠕動運動　）`を促進して便秘を予防するほか、大腸がんの予防や血中の`（　コレステロール　）`値を低下させる効果が期待できます。  
3. 糖質が体内でエネルギーに変換される際に、補酵素として不可欠なビタミンは`（　ビタミンB1　）`です。これが不足すると、疲れやすくなる原因になります。

#### **でんぷんについて**

1. 生のでんぷんに水を加えて加熱すると、のり状になって消化しやすくなります。この変化をでんぷんの`（　糊化　）`といいます。  
2. 糊化したでんぷんを冷たい場所に放置すると、水分が抜けて生のでんぷんに近い状態に戻り、味が落ちてしまいます。この現象をでんぷんの`（　老化　）`といいます。  
3. もち米のでんぷんは、粘りが非常に強い`（　アミロペクチン　）`という成分だけで構成されているため、もちもちとした食感になります。  
4. 私たちが普段食べているうるち米のでんぷんには、粘りの弱い`（　アミロース　）`が約20%含まれています。

#### **小麦粉について**

1. 小麦粉に水を加えてこねると、たんぱく質の一種である`（　グリアジン　）`と`（　グルテニン　）`が絡み合い、粘りと弾力性のある`（　グルテン　）`が形成されます。  
2. たんぱく質の含有量が11%〜12.5%と高い小麦粉は`（　強力粉　）`とよばれ、パンやパスタなど、しっかりとしたコシが必要な食品に使われます。

#### **いも類について**

1. じゃがいもの芽や緑色に変色した皮の部分には、`（　ソラニン　）`という有毒物質が含まれているため、調理の際には必ず取り除きましょう。  
2. さつまいもには、便通を促す効果がある`（　ヤラピン　）`という成分が含まれています。  
3. やまいもは、でんぷんを分解する消化酵素である`（　アミラーゼ　）`を豊富に含むため、生で食べても消化が良いという特徴があります。

\--------------------------------------------------------------------------------

炭水化物と同じく、脂質もまた重要なエネルギー源ですが、その性質や体での使われ方は大きく異なります。

## **3\. 脂質**

このセクションでは、炭水化物よりもエネルギー効率が高い脂質について学びます。脂質は細胞膜の構成成分やホルモンの材料にもなる重要な栄養素です。特に、体内で合成できず食事から摂取する必要がある「必須脂肪酸」の重要性や、脂質のバランスが健康に与える影響について理解を深めましょう。

#### **脂質の種類と働き**

1. 脂質は1gあたり約`（　9　）`kcalと、炭水化物やたんぱく質の2倍以上のエネルギーを発生させることができます。  
2. 脂質を構成する脂肪酸は、分子構造に二重結合がない`（　飽和脂肪酸　）`と、二重結合がある`（　不飽和脂肪酸　）`に大別されます。  
3. 体内で合成することができず、食物から摂取する必要がある脂肪酸を`（　必須脂肪酸　）`といい、代表的なものに`（　リノール酸　）`や`（　α-リノレン酸　）`があります。  
4. コレステロールは、ホルモンや`（　胆汁酸　）`の原料となる重要な物質ですが、過剰に摂取すると`（　動脈硬化　）`の原因となる可能性があります。  
5. マーガリンやショートニングを製造する過程で、`（　トランス脂肪酸　）`という、健康への影響が懸念される脂肪酸が生成されることがあります。

#### **脂肪酸の分類**

| 分類 | 名称 | 主な役割 | 多く含む食品 | 常温での状態 |
| :---- | :---- | :---- | :---- | :---- |
| 飽和脂肪酸 | パルミチン酸など | 主に体に蓄えられる脂肪になる | `（　食肉、バターなど　）` | `（　固体　）` |
| 一価不飽和脂肪酸 | `（　オレイン酸　）` | \- | オリーブオイルなど | 液体 |
| 多価不飽和脂肪酸 | `（　リノール酸　）`系 | 生命活動に必要な物質になる | だいず油など | `（　液体　）` |
| 多価不飽和脂肪酸 | `（　α-リノレン酸　）`系 | 生命活動に必要な物質になる | `（　アマニ油など　）` | 液体 |

\--------------------------------------------------------------------------------

エネルギー源となる栄養素の次は、私たちの体そのものを作り上げる主要な材料となる栄養素、たんぱく質について見ていきましょう。

## **4\. たんぱく質**

このセクションでは、筋肉や血液、皮膚など、体を構成する主成分であるたんぱく質について学びます。たんぱく質は約20種類のアミノ酸からできており、その組み合わせによって栄養価が決まります。また、卵のように加熱や攪拌で性質が変化する調理性の高さも、たんぱく質の特徴です。

#### **たんぱく質の構造と働き**

1. たんぱく質は、約20種類の`（　アミノ酸　）`が鎖のように多数結合してできています。  
2. 体内で合成することができず、食事から必ず摂取しなければならない9種類のアミノ酸を`（　必須アミノ酸　）`といいます。  
3. 食品に含まれるたんぱく質の栄養的な価値は、必須アミノ酸のバランスを示した`（　アミノ酸価　）`という指標で評価されます。  
4. ある食品において、理想的な量に対して最も不足している必須アミノ酸のことを`（　第一制限アミノ酸　）`といいます。  
5. 米のようにアミノ酸価が低い食品でも、アミノ酸価の高い肉や魚と一緒に食べることで、食事全体としての栄養価を高めることができます。これをたんぱく質の`（　補足効果　）`といいます。

#### **卵の調理性**

1. 卵白を泡立て器で攪拌すると、空気を抱き込んでふわふわのメレンゲになります。この性質を`（　起泡性　）`といいます。  
2. 卵黄に含まれる`（　レシチン　）`という成分には、本来混ざり合わない水と油を結びつける働きがあります。この働きを`（　乳化性　）`といい、マヨネーズ作りに利用されています。


たんぱく質は体の組織を作りますが、骨や歯の形成、そして体の様々な機能を調節するためには、無機質も欠かせません。

## **5\. 無機質**

このセクションでは、骨や歯の形成、神経の伝達、筋肉の収縮など、体の生理機能を調節するのに不可欠な無機質（ミネラル）について学びます。現代の食生活では、特定の無機質が不足したり、逆に過剰になったりする傾向があります。健康維持のために、それぞれの無機質の役割とバランスの重要性を理解しましょう。

#### **無機質の働き**

1. 現代の日本人の食生活では、`（　カルシウム　）`と`（　鉄　）`が不足しやすく、一方で食塩に含まれる`（　ナトリウム　）`は過剰摂取になりがちです。  
2. ほうれんそうなどに多く含まれる`（　シュウ酸　）`という成分は、カルシウムの吸収を妨げる性質があります。  
3. 鉄が不足すると、血液中のヘモグロビンが減少し、`（　鉄欠乏性貧血　）`を引き起こします。動物性食品に含まれる吸収されやすい鉄のことを`（　ヘム鉄　）`といいます。  
4. ナトリウムの過剰な摂取は、`（　高血圧　）`の主要な原因の一つとされています。

#### **無機質の分類**

| 分類 | 名称 | 主な働き | 欠乏症 | 多く含む食品 |
| :---- | :---- | :---- | :---- | :---- |
| 多量ミネラル | カルシウム(Ca) | 骨や歯の`（　形成　）`、筋肉の収縮、血液の凝固作用を助ける | 骨粗しょう症など | 牛乳・乳製品・小魚 |
| 多量ミネラル | カリウム(K) | 過剰なナトリウムを排出する | `（　疲労感　）` | 野菜・果実・海藻 |
| 多量ミネラル | ナトリウム(Na) | 細胞の浸透圧の調節 | 疲労感 | `（　食塩・みそ・しょうゆ　）` |
| 微量ミネラル | 鉄(Fe) | 血液の成分 | `（　鉄欠乏性貧血　）` | レバー・食肉・魚介 |
| 微量ミネラル | ヨウ素(I) | `（　ホルモンの成分　）` | 甲状腺肥大 | 海藻類 |

\--------------------------------------------------------------------------------

無機質と同様に、体の調子を整える働きを持つ栄養素としてビタミンがあります。これらはお互いに協力し合って機能することもあります。

## **6\. ビタミン**

このセクションでは、量はわずかでも体の機能を正常に保つために不可欠なビタミンについて学びます。ビタミンは油に溶ける「脂溶性ビタミン」と水に溶ける「水溶性ビタミン」に大別され、それぞれ性質が異なります。種類ごとの働きや、不足した場合に起こる欠乏症を理解し、毎日の食事でしっかり摂取することの重要性を確認しましょう。

#### **ビタミンの分類と特性**

1. ビタミンは、油に溶けやすく、油脂と一緒に摂ると吸収率が高まる`（　脂溶性ビタミン　）`と、水に溶けやすい`（　水溶性ビタミン　）`に分けられます。  
2. 脂溶性ビタミンは体内に蓄積されやすいため、サプリメントなどで過剰に摂取すると`（　過剰症　）`を引き起こす可能性があり、注意が必要です。  
3. 緑黄色野菜に多く含まれるβ-カロテンのように、体内でビタミンAに変換されて作用する物質を`（　プロビタミンA　）`といいます。  
4. ビタミンDは、食品から摂取するだけでなく、日光の`（　紫外線　）`を浴びることで私たちの皮膚でも合成されます。  
5. ビタミンCは、皮膚や血管を丈夫に保つ`（　コラーゲン　）`の合成に不可欠な栄養素です。

#### **ビタミンの分類**

| 分類 | 名称 | 主な働き | 欠乏症 | 多く含む食品 |
| :---- | :---- | :---- | :---- | :---- |
| 脂溶性 | ビタミンA | 皮膚・粘膜の保護、視力調節 | 発育障害、皮膚乾燥、`（　夜盲症　）` | レバー・緑黄色野菜 |
| 脂溶性 | ビタミンD | カルシウムやリンの吸収を調節 | くる病、骨軟化症 | `（　魚類・卵黄・きのこ類　）` |
| 脂溶性 | ビタミンK | `（　血液凝固　）` | 血液の凝固不良 | 緑黄色野菜・納豆 |
| 水溶性 | ビタミンB1 | 糖質の代謝に関係 | 疲労感、食欲不振、`（　脚気　）` | 豚肉・豆・卵黄 |
| 水溶性 | ビタミンB2 | 発育促進、皮膚・粘膜の保護 | 口内炎・皮膚炎 | `（　レバー・チーズ・牛乳　）` |
| 水溶性 | ビタミンC | `（　コラーゲン　）`の合成を助け、結合組織を健全に保つ | 壊血病 | 野菜・果物・いも類 |

\--------------------------------------------------------------------------------

ここまで5大栄養素のそれぞれの働きについて学んできました。次のセクションでは、これらの知識を活かして、日々の食事をどのように計画すればよいか、その具体的な方法を見ていきましょう。

## **7\. 食事計画とバランス**

このセクションでは、健康を維持・増進するために、これまで学んだ栄養素の知識を実際の食事計画にどう活かすかを学びます。国が示す摂取基準や、栄養バランスを視覚的に理解するためのツールなどを知ることで、自分や家族のための、より具体的で健康的な食生活を設計できるようになります。

1. 健康な人々を対象に、エネルギーや各栄養素について、一日にどれくらい摂取すればよいかの基準を示したものを「`（　日本人の食事摂取基準　）`」といいます。  
2. 様々な食品について、食べられる部分（可食部）100gあたりにどの栄養素がどれくらい含まれているかを示した一覧表を「`（　日本食品標準成分表　）`」といいます。  
3. 食品を栄養的な特徴から「血や肉をつくる（赤）」「体の調子をよくする（緑）」「力や体温となる（黄）」の3つのグループに分けたものを`（　3色食品群　）`といいます。  
4. 1日に「何を」「どれだけ」食べたらよいかの目安を、主食・主菜・副菜などの料理グループに分け、コマのイラストで分かりやすく示したものを`（　食事バランスガイド　）`といいます。

栄養バランスの取れた食事を計画することも大切ですが、その食事が安全でなければ意味がありません。最後のセクションでは、食の安全を守るための基本である食中毒予防について学びます。

## **8\. 食中毒の予防**

このセクションでは、安全な食事に不可欠な食中毒予防について学びます。食中毒の原因となる細菌やウイルスの種類とそれぞれの特徴、そしてそれらを防ぐための基本的な三原則を理解することは、自分自身や家族を食中毒から守るために非常に重要です。

#### **食中毒の種類**

1. 生の鶏卵や、犬・猫などのペットから感染することがある食中毒菌は`（　サルモネラ　）`です。  
2. 鶏肉に付着していることが多く、比較的少ない菌数でも食中毒を発症させることがあるのは`（　カンピロバクター　）`です。  
3. ベロ毒素とよばれる非常に強い毒素を産生し、重症化すると後遺症が残ることもあるのは`（　病原性大腸菌O-157　）`です。  
4. 人の手指の傷などから食品に付着しやすく、この菌が作る毒素は加熱しても分解されにくい特徴を持つのは`（　黄色ブドウ球菌　）`です。  
5. 冬場に流行しやすく、特に牡蠣などの二枚貝が原因となり、大規模な集団感染を引き起こすことがあるのは`（　ノロウイルス　）`です。  
6. さばやいかなどの魚介類に寄生しており、生で食べると激しい腹痛を引き起こす寄生虫は`（　アニサキス　）`です。

#### **食中毒予防の三原則**

1. 食中毒予防の三原則は、菌を「`（　つけない　）`」、菌の増殖を防ぐために「`（　増やさない　）`」、そして加熱などによって菌を「`（　やっつける　）`」ことです。  
2. 菌を増やさないためには、食品を低温で保存することが効果的です。多くの食中毒菌は`（　10　）`℃以下ではほとんど増殖できなくなります。  
3. 菌をやっつける（殺菌する）ためには、食品の中心部の温度を`（　75　）`℃以上で`（　1　）`分間以上、十分に加熱することが必要です。  
4. 食品の製造工程において、危害の原因となりうる要因を分析し、それを管理することで食品の安全性を確保する衛生管理の手法を`（　HACCP　）`（ハサップ）といいます。
